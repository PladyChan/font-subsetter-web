# TypeTrim - Smart Font Subsetting Tool

一个智能的字体子集化工具，帮助优化字体文件大小。

## 功能特点

- 支持 TTF/OTF 字体文件
- 智能保留必要字符
- 支持多种字符集选项
- 在线即时处理

## 使用方法

1. 访问 [TypeTrim](https://font-subsetter-web.vercel.app)
2. 上传字体文件
3. 选择需要保留的字符集
4. 下载优化后的字体文件
