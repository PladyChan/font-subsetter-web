## 📖 什么是 TrimType？

TrimType 是一个**智能字体裁剪工具**，帮助开发者、设计师和内容创作者优化字体文件大小，提升网站性能和用户体验。

### 核心功能

通过智能分析字体文件，TrimType 可以：
- ✂️ **精准裁剪**：只保留你实际使用的字符，大幅减小字体文件体积
- 🎯 **智能识别**：自动识别并保留必要的字符集（中文、英文、数字、标点等）
- ⚡ **即时处理**：在线处理，无需安装，上传即用
- 📦 **多格式支持**：支持 TTF、OTF、WOFF、WOFF2、EOT、TTC 等主流字体格式

---

## ✨ 特色功能

### 1. 🎨 智能字符集选择

- **中文字符**：支持全部常用中文字符（CJK 统一表意文字）
- **英文字母**：完整保留 a-z、A-Z
- **数字与标点**：支持中英文标点、数学符号、货币符号等
- **特殊字符**：连字、分数、上标下标、变音符号等
- **自定义字符**：支持手动输入需要保留的特殊字符

### 2. 🔄 可变字体支持

- **完整保留字重变化**：自动检测可变字体，保留所有字重变化能力
- **智能优化**：非可变字体自动删除冗余表，进一步减小文件大小
- **完美兼容**：处理后的字体保持原有格式和兼容性

### 3. 🚀 极速处理体验

- **零配置**：无需安装，打开即用
- **拖拽上传**：支持拖拽文件或点击上传
- **批量处理**：可同时处理多个字体文件
- **自动下载**：处理完成后自动触发下载

### 4. 📊 详细处理报告

- **文件大小对比**：显示原始大小和优化后大小
- **压缩比例**：清晰展示文件减少的百分比
- **处理状态**：实时显示处理进度和结果

## 🚀 快速开始

### 在线使用

直接访问部署后的网站，无需任何安装配置。

### 本地开发

1. **克隆项目**
   ```bash
   git clone https://github.com/PladyChan/font-subsetter-web.git
   cd font-subsetter-web
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **启动服务**
   ```bash
   # Mac 用户
   ./启动本地服务.command
   
   # 或手动启动
   python3 app.py
   ```

4. **访问网站**
   打开浏览器访问：`http://localhost:5000`

## 📝 使用说明

### 基本流程

1. **上传字体文件**
   - 拖拽字体文件到上传区域，或点击选择文件
   - 支持同时上传多个文件

2. **选择字符集**
   - 勾选需要保留的字符类型（英文字母、中文字符、数字、标点等）
   - 可添加自定义字符

3. **开始处理**
   - 点击"开始处理"按钮
   - 等待处理完成（通常只需几秒钟）

4. **下载结果**
   - 处理完成后自动下载优化后的字体文件
   - 查看文件大小对比和压缩比例

### 使用建议

- **Web 字体优化**：只选择网站实际使用的字符集
- **移动端优化**：优先选择必要的字符，最大化压缩效果
- **多语言网站**：根据页面语言选择对应的字符集
- **图标字体**：使用自定义字符功能保留特定图标
