# TrimType 使用指南

## 📖 这是什么？

TrimType 是一个在线字体裁剪工具，可以帮你：
- 减小字体文件大小（通常可减少 60-90%）
- 提升网站加载速度
- 节省带宽和存储空间

**原理**：删除字体中不需要的字符，只保留你实际使用的部分。

---

## 🚀 如何使用

### 方式一：在线使用（推荐）

直接访问部署后的网站，无需安装任何软件。

### 方式二：本地使用

如果你想在本地运行 TrimType，可以使用一键启动脚本：

#### Mac 用户

1. **双击启动**
   - 找到项目目录中的 `👉启动本地服务.command` 文件
   - 双击运行，脚本会自动：
     - 检查 Python 环境
     - 安装所需依赖
     - 启动本地服务
     - 自动打开浏览器

   ⚠️ **如果遇到 macOS 安全警告**（"无法验证开发者"）：
   
   这是 macOS 的安全机制，因为文件是从网络下载的。解决方法：
   
   **方法 1：右键打开（推荐）**
   - 右键点击 `👉启动本地服务.command` 文件
   - 选择"打开"
   - 在弹出的对话框中点击"打开"
   - 之后就可以直接双击运行了
   
   **方法 2：终端命令**
   - 打开终端，运行以下命令：
     ```bash
     cd "项目目录路径"
     xattr -c "👉启动本地服务.command"
     ```
   - 然后就可以正常双击运行了
   
   **方法 3：系统设置**
   - 打开"系统设置" > "隐私与安全性"
   - 在"安全性"部分找到相关提示
   - 点击"仍要打开"或"允许"

2. **访问地址**
   - 服务启动后会自动打开浏览器
   - 或手动访问：`http://localhost:8888`

3. **停止服务**
   - 在终端窗口中按 `Ctrl+C` 停止服务
   - 或直接关闭终端窗口

#### Linux/Windows 用户

1. **运行启动脚本**
   ```bash
   chmod +x run_local.sh
   ./run_local.sh
   ```

2. **访问地址**
   - 浏览器访问：`http://localhost:8888`

3. **停止服务**
   - 在终端窗口中按 `Ctrl+C` 停止服务
   - 或直接关闭终端窗口

**提示**：首次运行会自动安装依赖，可能需要几分钟时间。

## 📝 使用步骤

### 1. 上传字体文件
- 拖拽文件到上传区域，或点击选择文件
- 支持 TTF、OTF、WOFF、WOFF2、EOT、TTC 格式
- 可同时上传多个文件

### 2. 选择字符集
根据需求勾选需要保留的字符：

**基础选项**：
- ✅ 英文字母 (a-z, A-Z)
- ✅ 中文字符（全部）
- ✅ 英文标点 (,.!?@#$%等)
- ✅ 中文标点（，。！？……）
- ✅ 数字 (0-9)

**扩展选项**（按需选择）：
- 数学符号、货币符号、箭头、版权符号
- 连字、分数、上标下标、变音符号

**自定义字符**：
- 在输入框中手动输入需要保留的特殊字符（无需分隔）

### 3. 开始处理
点击"开始处理"按钮，等待几秒钟即可完成。

### 4. 下载结果
处理完成后会自动下载优化后的字体文件。

## 💡 使用建议

- **纯英文网站**：只选英文字母、英文标点、数字
- **中文网站**：选英文字母、中文字符、中英文标点、数字
- **最大化压缩**：只选择实际使用的字符类型
- **可变字体**：自动保留所有字重变化，无需特殊操作

## ❓ 常见问题

**Q: 处理后的字体还能正常使用吗？**  
A: 可以，只删除未选择的字符，保留的字符功能正常。

**Q: 会改变字体格式吗？**  
A: 不会，保持原有格式。

**Q: 文件能减少多少？**  
A: 通常可减少 60-90%，取决于选择的字符集。

**Q: 处理失败怎么办？**  
A: 检查文件格式和大小（不超过 100MB），确保至少选择一个字符集，然后重试。
